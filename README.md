#### 简介
中国象棋控件。

- [项目主页](http://kanglin.github.io/ChineseChessControl/)
- [项目位置](https://github.com/KangLin/ChineseChessControl)

##### 功能
- [x] 中国象棋 ActivX 控件
- [ ] Qt 中国象棋控件
- [ ] 中国象棋程序
  + [ ] 单机版本
  + [ ] 联网版本
